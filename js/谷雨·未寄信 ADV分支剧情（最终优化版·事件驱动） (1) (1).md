# 谷雨·未寄信 ADV分支剧情（最终优化版·事件驱动）

## 核心设定

- 全剧分支由\*\*事件选择\*\*驱动，而非性格倾向标签，跳转逻辑完全贴合“选择的行为结果”，无生硬人设绑定

- 每幕4个选项，打破二元对立设计，补充“第三种/第四种事件走向”（如“是否停留”外新增“前往其他地点”），兼顾选择多样性与剧情合理性

- 累计经历10\-14幕即可解锁结局，严格遵循时间线逻辑（清晨→上午→午休→下午→黄昏→夜晚/次日），分支间可灵活跳转，无并列场景冲突

- 全程自然覆盖关键词：谷雨、工位、镜子、跑步、公路旅行、猫、七里香、三分糖、第二杯半价，不刻意堆砌

- 基调：温柔治愈、自我成长，所有分支均指向积极的心意安放与自我接纳，结局根据累计事件选择的走向自然生成

- 幕间差分：同一幕内根据前序选择的细微差异（如通勤是否拖延），优化场景细节描述，不新增幕数、不改变选项及跳转逻辑，仅体现状态区分

第一幕：谷雨出门・风启初心（必选幕・清晨）
【剧情过渡】
你轻轻合上家门，谷雨清晨的风扑面而来，带着泥土与青草的湿润香气，路面还留着细雨打过的浅痕，枝叶上的水珠偶尔滴落，砸在地面漾开细小的涟漪。
心底那封未寄出的信，随着脚步轻轻发烫。耳机里随机播放的音乐，忽然响起《七里香》的前奏，少年时操场奔跑的晚风、课间分享的三分糖奶茶、约定好的公路旅行，全都随着旋律浮现在眼前。
【关键抉择 1】
A 加快脚步直奔工位，不想耽误晨间的工作节奏
→ +J；直接进入第三幕（工位场景・准时抵达，从容有序）
B 放慢脚步走老街支路，想顺路看看谷雨的街边景致
→ +P、+S；进入第二幕（遇猫支线），后续抵达工位略有延迟
C 拐进街角的便利店，买一杯热饮再继续赶路
→ +F、+S；买完热饮后，顺路走老街进入第二幕，后续抵达工位略有延迟

→ +I、+N；歌曲结束后，选择直奔工位进入第三幕（工位场景・轻微延迟，略显仓促）
